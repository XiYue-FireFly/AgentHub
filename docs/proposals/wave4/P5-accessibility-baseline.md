# P5 无障碍与 i18n 基线 — 规格草案

**代号：** `W4-A11Y`  
**状态：** 草案  
**参考：** WCAG 2.2 AA（目标，可分期）  

## 1. 目标

保证 AgentHub **关键路径**可键盘完成，焦点可见，中英切换稳定（i18n 已有基础）。

## 2. 关键路径（必须闭环）

1. 打开工作台 → 新建会话  
2. Composer 输入并发送  
3. 打开设置 → 切换语言 / 主题  
4. 打开 Git 面板 → 刷新  
5. 打开终端 → 新建 tab  

## 3. 检查清单（MVP）

- [ ] 所有可点击控件可 Tab 到达  
- [ ] 焦点环对比度可见（非 `outline: none` 无替代）  
- [ ] 图标按钮有 `aria-label` / `title`  
- [ ] 模态框 Esc 关闭且焦点陷阱  
- [ ] 动态通知可被读屏感知（或提供文本区）  

## 4. i18n

- 已有：`setLang` + App `key={lang}` 重挂载  
- Wave4：扫工作台硬编码英/中字符串，统一 `tr()`  
- 禁止新增仅中文或仅英文的用户可见文案  

## 5. 验收

- 人工键盘走完关键路径无死锁  
- 文档化已知例外列表（若有）
